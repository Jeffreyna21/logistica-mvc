# Logística MVC — Express.js + Node.js

Aplicación que calcula el costo total de envíos por repartidor
en un rango de fechas, usando el patrón MVC con Express.js.

## MVC utilizado
Express.js + EJS (Node.js)

## Descripción
El sistema filtra envíos por fecha y calcula el costo total
por repartidor aplicando la tarifa por kg de cada zona.

## Cómo correrlo localmente
1. Clonar el repositorio
2. Ejecutar `npm install`
3. Ejecutar `node database.js` (solo la primera vez)
4. Ejecutar `node app.js`
5. Abrir http://localhost:3000
